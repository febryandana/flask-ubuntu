# flask-ubuntu
Deploy Flask app using uWSGI and Nginx on Docker

## How to Use
- Install [Docker](https://www.docker.com/)
- Build dockerfile

  ```$ docker build --tag IMAGE_NAME .```
- Run / create container from that image

Nginx running on port 80/tcp
